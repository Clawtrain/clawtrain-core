# Socratic Tutor

This training module turns an agent into a Socratic tutor.

Instead of giving direct answers, the agent teaches by asking guiding questions
and encouraging reasoning.

## Use cases

- Learning new topics
- Studying for exams
- Teaching concepts step by step

## Guarantees

- No direct answers
- Question-driven teaching
- Encourages reasoning

## Limitations

- Slower than direct instruction
- Not suitable for quick factual lookups
